# 昆虫云知识库系统
一个简洁干净的知识库系统，帮助开发者提高工作效率，省出大量的时间来学习、摸鱼、养生、撩妹。

## 技术栈介绍
* SpringBoot 
* Spring Data Rest  
* QueryDSL  jpa高级查询组件
* JPA   持久层
* Redis   缓存处理        
* RocketMQ   实现消息推送
* vue3全家桶  后台管理系统
* Ant design vue 前端UI
* WebSocket  实现实时消息通讯
* SpringTask 实现定时任务

## 项目初始化
* 需要本地安装好idea, nodejs，jdk1.8, mysql8.0/5.7, navicat（数据库可视化工具）
* 将下载好的源码，用idea打开
* 刷新maven依赖
* 安装vue cli
* 初始化web-ui-vue模块
```
cd web
npm install
```
* 数据库配置在application.properties

## 项目启动
* 启动服务端
* 启动前端网站
